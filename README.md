# Ofertas de Juegos PC 🎮

Aplicación web desarrollada con Vue.js que consume datos desde la API pública CheapShark para mostrar ofertas de videojuegos de PC.

## 🔗 API Utilizada

CheapShark API  
https://www.cheapshark.com/api

Endpoint utilizado:
https://www.cheapshark.com/api/1.0/deals?storeID=1&pageSize=30

## 🛠 Tecnologías

- Vue.js
- Vite
- HTML5
- CSS3
- JavaScript
- Fetch API
- Vercel (deploy)

## 💡 Funcionalidades

- Consumo de API pública
- Visualización de juegos en tarjetas
- Mostrar precio normal y precio en oferta
- Mostrar porcentaje de descuento
- Buscador por nombre
- Ordenar por:
  - Nombre
  - Precio
  - Mayor descuento
- Diseño responsive (PC, tablet y móvil)
